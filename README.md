# 🌐 Streamlit Projects

This repository contains interactive web applications built using Streamlit.  
The focus is on transforming data science and machine learning projects into user-friendly web apps.

---

## 🚀 Objectives

- Build interactive web applications using Streamlit  
- Deploy data science and machine learning models  
- Create intuitive user interfaces for data exploration  
- Share projects in a visually appealing and accessible way  

---

## 🛠️ Tech Stack

- Python 🐍  
- Streamlit  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib / Seaborn
